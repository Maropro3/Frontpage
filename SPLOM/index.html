Finding groups of exoplanets with clustering


The NASA Exoplanet Archive contains over 4000 confirmed exoplanets. These vary widely in terms of orbit, mass, radius and other parameters. Using
different clustering techniques, we can try to extract some patterns in the raw data and create some grouping or categories to label these exoplanets.
